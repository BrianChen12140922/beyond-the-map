Project rationale

A cultural tourism website built around interaction.

The website helps visitors discover Hong Kong hiking routes through local myth, landscape form, and short practical route information.

Objective

Map-led storytelling

The homepage places the interactive map at the centre so tourists can explore by trail, district, and myth instead of reading a static essay.

Target audience

Cultural hikers

The primary users are visitors who want a light outdoor itinerary with hidden culture, local stories, and clear route choices.

Interaction

Click to reveal

The map, route buttons, and myth cards give the project enough JavaScript interaction for a working web prototype.

Submission format

Static web build

The site uses HTML, CSS, and JavaScript files that can be uploaded directly to GitHub Pages for the working website URL.

Academic integrity

References

Route facts and cultural-event details are attributed below. Any additional external images or AI-generated assistance should be acknowledged in the final submission package.

Demo path

Presentation flow

Start at the homepage, click the map, open the full map page, then show Explore, Myths, and responsive mobile layout.
